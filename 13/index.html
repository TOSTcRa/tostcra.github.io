Фонове відео

На цій сторінці використовується фонове відео. Воно розташоване за контентом і автоматично відтворюється без звуку.

<div class="video-background">
  <video autoplay muted loop playsinline>
    <source src="video.mp4" type="video/mp4">
  </video>
</div>
Атрибути: autoplay (автовідтворення), muted (без звуку), loop (повтор), playsinline (для мобільних)

Відео через тег <video>

Вбудований HTML5 відеоплеєр з елементами керування:

Базовий приклад

Відео з автовідтворенням (muted)

<video controls poster="preview.jpg">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
</video>
Атрибути video: controls, autoplay, muted, loop, poster, preload, width, height

Аудіо через тег <audio>

HTML5 аудіоплеєр:

Базовий аудіоплеєр

Аудіо з автовідтворенням

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
</audio>
Атрибути audio: controls, autoplay, muted, loop, preload (none, metadata, auto)

Карта зображення <map> та <area>

Клікабельні області на зображенні:

Області:
• Верхня ліва частина (rect) → Відео
• Верхня центральна (rect) → Аудіо
• Верхня права (rect) → Карти
• Центральне коло (circle) → Canvas
• Лівий нижній трикутник (poly) → SVG
<img src="image.jpg" usemap="#mymap">
<map name="mymap">
  <area shape="rect" coords="x1,y1,x2,y2" href="link">
  <area shape="circle" coords="x,y,radius" href="link">
  <area shape="poly" coords="x1,y1,x2,y2,x3,y3" href="link">
</map>
