Властивості animation

animation-name

Назва @keyframes анімації

animation-duration

Тривалість анімації (наприклад: 2s, 500ms)

animation-timing-function

Функція часу: ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier()

animation-delay

Затримка перед початком анімації

animation-iteration-count

Кількість повторів: число або infinite

animation-direction

Напрямок: normal, reverse, alternate, alternate-reverse

animation-fill-mode

Стан до/після: none, forwards, backwards, both

animation-play-state

Стан відтворення: running, paused

Скорочений запис:

animation: name duration timing-function delay iteration-count direction fill-mode;

@keyframes приклади

Проста анімація (from/to):

@keyframes slideIn {
    from {
        transform: translateX(-100%);
        opacity: 0;
    }
    to {
        transform: translateX(0);
        opacity: 1;
    }
}

Анімація з відсотками:

@keyframes bounce {
    0%, 100% {
        transform: translateY(0);
    }
    25% {
        transform: translateY(-20px);
    }
    50% {
        transform: translateY(0);
    }
    75% {
        transform: translateY(-10px);
    }
}

Анімація кількох властивостей:

@keyframes complexEffect {
    0% {
        transform: scale(1) rotate(0deg);
        background-color: #e94560;
        border-radius: 10px;
    }
    50% {
        transform: scale(1.2) rotate(180deg);
        background-color: #00fff5;
        border-radius: 50%;
    }
    100% {
        transform: scale(1) rotate(360deg);
        background-color: #e94560;
        border-radius: 10px;
    }
}
