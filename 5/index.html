Маркировані списки (ul)

Стандартні маркери

disc (за замовчуванням):

  • Перший елемент
  • Другий елемент
  • Третій елемент

circle:

  • Перший елемент
  • Другий елемент
  • Третій елемент

square:

  • Перший елемент
  • Другий елемент
  • Третій елемент

none (без маркера):

  • Перший елемент
  • Другий елемент
  • Третій елемент

Нумеровані списки (ol)

Різні типи нумерації

  1. Перший
  2. Другий
  3. Третій

decimal-leading-zero (01, 02, 03...):

  1. Перший
  2. Другий
  3. Третій

lower-alpha (a, b, c...):

  1. Перший
  2. Другий
  3. Третій

upper-alpha (A, B, C...):

  1. Перший
  2. Другий
  3. Третій

lower-roman (i, ii, iii...):

  1. Перший
  2. Другий
  3. Третій

upper-roman (I, II, III...):

  1. Перший
  2. Другий
  3. Третій

lower-greek (α, β, γ...):

  1. Перший
  2. Другий
  3. Третій

georgian:

  1. Перший
  2. Другий
  3. Третій

Атрибути ol

start="5" (починається з 5):

  1. П'ятий елемент
  2. Шостий елемент
  3. Сьомий елемент

reversed (зворотній порядок):

  1. Перший (буде 3)
  2. Другий (буде 2)
  3. Третій (буде 1)

value на окремому елементі:

  1. Перший
  2. Десятий
  3. Одинадцятий

list-style-position

outside (за замовчуванням):

  1. Маркер знаходиться зовні блоку тексту. Це стандартна поведінка.
  2. Другий елемент

inside:

  1. Маркер знаходиться всередині блоку тексту. Текст обтікає маркер.
  2. Другий елемент

Списки визначень (dl)

Простий список визначень

HTML
